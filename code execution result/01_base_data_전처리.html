문제출제 목적¶

  • 내부요인(상품력, 시즌별 상품변화) + 외부요인(날씨,이슈,시청률) 을 통해 프로그램 매출 실적을 사전에 예측하고 대응

데이터 설명¶

  • 판매가 0원(무형, 보험 상품)은 추정 제외
  • 매주 토요일 18:00 ~ 18:20은 제외(이미 제외되어 있음)
  • 편성시간은 10~60분 내외
  • 편성표는 6:20~2:20(익일)으로 구성(심야시간대 제외)
  • 취급액 = 판매단가 * 주문량

def make_season(x):
    if 3 <= x <= 5 :
        return('봄')
    elif 6 <= x <= 8 :
        return('여름')
    elif 9 <= x <= 11 :    
        return('가을')
    else :
        return('겨울')
